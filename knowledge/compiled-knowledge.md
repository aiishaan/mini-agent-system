# Compiled Knowledge

## Summary
Compiled knowledge is the practice of turning repeated work into durable local notes,
links, summaries, and topic pages instead of treating every question as a throwaway
retrieval event.

## Key Points
- Local Markdown notes are easy to inspect and edit.
- Wiki links create a graph between related ideas.
- Agents can improve the knowledge base by updating notes as part of the workflow.
