# Agentic AI

## Summary
Agentic AI describes software that uses goal-directed agents to plan, take actions,
inspect results, and coordinate work across a workflow.

## Key Points
- A master agent can orchestrate smaller specialist agents.
- Useful specialist roles include research, summarization, writing, linking, and validation.
- A simple agentic system should make each step visible enough to debug.
